2016-2017 小结

Jun 12 2017

时光如水呐～ 眼瞅着又一年过去了，小结必不可少。

生活总是不经意间给我们惊喜、让我们成长。希望我们也可以年复一年，追求更美好的生活。

关于我的 2016 - 2017：

  1. 跳槽。进入猫厂新人培训的第一天我就想到会有这么一天，不过没想到这么快。在豚厂这一年里，确实更有好感。这里工作生活自由度更高，且以我爱折腾的本性，学到的东西也并不少。
  2. 第一次拿到薪水和年终之外的 bonus。
  3. 第一次不在自己老家过年。讲真，除夕很久没这么热闹过了。
  4. 买了人生中的第一套房子。
  5. 进而有了第一次入不敷出，靠信用卡度日的囧境。
  6. 第一次现场看亚冠，效果果然比在屏幕前屌好多。浩克牛逼！上港加油 : )
  7. 终于算是在 Github 上活跃起来了: 有了自己的 Organization; 有了第一次 PR; 开源了第一个练手的项目 WeVoting (虽然只有 20+ star，但代码规范、单元测试、覆盖率、持续集成等都在努力接近社区水平); 收到第一个 issue …
  8. 坚持更新技术博文，并逐渐对于数据敏感，有了推广意识。
  9. 入坑王者农药 （玩也很重要的好吗 =,=）。
  10. 未来几个月，又将跨越人生另一里程碑。和去年一样，愿爱和责任与我同在。

今天赶上肠胃不适，卧病在床，但头脑还算清醒，思维也仍然活跃。于是刚好腾出空来，完结此篇。

The end.

– 2017.06.14 中午于上海
